API

You can use this URL Shortening service by sending a get request to:

https://shor10.glitch.me/new/
Accepted query parameters: url, callback.

Example

Here is an example get URL:

https://shor10.glitch.me/new/url?=http://someDomain.com&callback=?

Make sure you include http:// or https:// in the url query.
Output

Example Output:

{
"_id": "5993a2238c07b33b9aaa272a",
"name": "a9in",
"full_url": "http://twitter.com",
"short_url": "https://shor10.glitch.me/r/a9in"
}
